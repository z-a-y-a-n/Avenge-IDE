# Avenge-IDE

Welcome to the Avenge-IDE repository! This project is developed by Zayan.U.

## Introduction

Avenge-IDE is a powerful, user-friendly Integrated Development Environment (IDE) designed with Python3. This IDE aims to provide a seamless coding experience, making it easier for developers to write, save, new file and run their code.

## Features

- **Ease of Use**: Avenge-IDE is designed with a focus on usability. Its intuitive interface allows you to navigate through your code effortlessly.
- **Python3 Support**: Built with Python3, Avenge-IDE supports Python development out of the box.
